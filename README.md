# Multi-Client Chat Application

## Overview
A real-time chat application using TCP socket programming that allows multiple clients to communicate through a server.

## Features
- Multi-client support
- Real-time messaging
- Client-server architecture
- Multithreading

## Tech Stack
- Python
- Socket Programming
- Threading

## How to Run
1. Run server:
   python server.py

2. Run multiple clients:
   python client.py

## Concepts Used
- TCP/IP Networking
- Socket Programming
- Concurrency
